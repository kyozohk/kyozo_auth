# **App Name**: Auth Starter

## Core Features:

- User Signup: Allow users to create a new account using their email and password and save into Firestore.
- User Login: Authenticate existing users using their email and password, securely validating credentials with Firestore.
- Password Reset: Enable users to reset their password via email using the provided Firebase reset link and Firestore authentication. Link: https://prod.api.kyozo.com/api/utils/reset-password-redirect?mode=action&oobCode=code.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to convey trust and security.
- Background color: Light gray (#F0F4F7) to provide a clean and neutral backdrop.
- Accent color: Subtle orange (#FFA726) for highlighting important actions and links.
- Body and headline font: 'Inter' sans-serif for a modern and readable user experience.
- Clean, centered layout with clear form fields and prominent action buttons for ease of use.
- Use simple, outlined icons for input fields and navigation to enhance usability.
- Subtle animations for form feedback (success, error) and transition effects to enhance user engagement.
